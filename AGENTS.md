# Project Instructions

## UI/UX Default Workflow

- For any UI/UX design, redesign, or frontend polish work in this project, use the local skill at `.codex/skills/ui-ux-pro-max/SKILL.md` as the primary design system and UX guidance source.
- Treat [Magic UI Showcase](https://magicui.design/showcase) as an approved inspiration/reference library for visual patterns, motion ideas, and component presentation.
- Before substantial UI work, generate or consult a design direction from `ui-ux-pro-max` first, then adapt implementation to the existing project structure and stack.
- Keep accessibility, responsive behavior, motion restraint, and clear interaction states aligned with the `ui-ux-pro-max` guidance.
- Reuse ideas inspired by Magic UI, but do not blindly copy patterns that conflict with this product's content, usability, or performance needs.
