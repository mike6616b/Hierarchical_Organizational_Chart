# UI/UX Resources

This project uses these two UI/UX references by default:

- Magic UI Showcase: https://magicui.design/showcase
- UI UX Pro Max local skill: `.codex/skills/ui-ux-pro-max/SKILL.md`

## Local Installation

`UI UX Pro Max` has been initialized for this project under:

```text
.codex/skills/ui-ux-pro-max/
```

Key files:

- `.codex/skills/ui-ux-pro-max/SKILL.md`
- `.codex/skills/ui-ux-pro-max/scripts/search.py`

## Suggested Usage

When starting a new UI direction, first consult the skill guidance and, if needed, use the design-system flow described in the skill.

Reference commands from the installed skill:

```bash
python3 .codex/skills/ui-ux-pro-max/scripts/search.py "organization chart dashboard" --design-system -p "Hierarchical Organizational Chart"
python3 .codex/skills/ui-ux-pro-max/scripts/search.py "organization chart interactions" --domain ux
python3 .codex/skills/ui-ux-pro-max/scripts/search.py "data visualization" --domain chart
```

Use Magic UI as a visual inspiration source for:

- motion and reveal patterns
- polished component presentation
- empty states, highlights, and premium-feel sections

Prefer adapting those ideas to the project's information architecture instead of copying showcase patterns verbatim.
